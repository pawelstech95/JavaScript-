UWAGI:

Instrukcje na temat tego, jak korzystać z plików źródłowych
znajdziesz w lekcji pt. "Jak korzystać z plików źródłowych".

Najważniejsze aby nie zapomnieć, że wymagane jest zainstalowane
środowisko Node.js. Następnie po przejściu w terminalu
do katalogu app/system należy wpisać:

> npm install

a pobrane zostaną wszystkie zależności z pliku package.json

Od teraz aplikację można uruchomić wpisując:

> npm start

a zatrzymać wciskając CTRL+C

W końcowych lekcjach kursu, gdzie pracujemy z npm i webpackiem,
na początku każdej lekcji należy przejść w terminalu
do odpowiedniego katalogu i wpisać:

> npm install

aby zainstalować wszystkie zależności z pliku package.json
